# ✧ Be True Studio ✧

Site institucional para estúdio de make e spa, com foco em identidade visual, experiência do usuário e fluxo de agendamento.

## Páginas »

- `/Home`
- `/sobre`
- `/portfolio`
- `/agendamento`
- `/contato`

## Rodar localmente »

```bash
npm install
npm run dev
```
